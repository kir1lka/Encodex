# Шифрование данных в файлах средствами C++

Десктопное приложение для шифрования и расшифровки данных, разработанное на C++. Программа поддерживает как симметричное (AES), так и асимметричное (RSA) шифрование, позволяет генерировать ключи и работать с файлами. Интуитивно понятный интерфейс делает процесс шифрования простым и доступным.

### Шифрование данных (AES/RSA)
Чтобы зашифровать данные, введите текст в поле "большой секрет в секрете", укажите ключ шифрования и нажмите кнопку "Зашифровать". Также можно загрузить файл для шифрования, нажав "Открыть файл".

### Расшифровка данных
Для расшифровки вставьте зашифрованный текст в соответствующее поле, укажите ключ и нажмите "Расшифровать". Приложение поддерживает автоматическое определение алгоритма расшифровки на основе предоставленного ключа.
