# Шифрование данных в файлах средствами C++

Данный проект представляет собой десктопное приложение, разработанное на C++, которое демонстрирует работу с криптографией. Программа позволяет шифровать и расшифровывать данные, используя как симметричные (AES), так и асимметричные (RSA) алгоритмы, а также управлять криптографическими ключами.

## Скриншоты интерфейса

Здесь представлен внешний вид приложения и его основные функциональные возможности.

### Главное окно: Шифрование и расшифровка

На первом экране представлен базовый интерфейс для работы с шифрованием. Пользователь может ввести текст (или загрузить файл), указать ключ и выполнить операцию.

[![Интерфейс шифрования и расшифровки](https://ltdfoto.ru/images/n5gPOj.png)](https://ltdfoto.ru/image/n5gPOj)
*На скриншоте показаны поля для ввода текста, выбора файла, а также кнопки "Зашифровать" и "Расшифровать". Режимы AES и RSA переключаются в верхней панели навигации.*

### Управление ключами RSA

Вторая вкладка или часть интерфейса отвечает за асимметричное шифрование RSA. Здесь пользователь может видеть сгенерированные публичный и приватный ключи, а также использовать их для шифрования и расшифровки данных.

[![Генерация и управление RSA ключами](https://ltdfoto.ru/images/n5gVji.png)](https://ltdfoto.ru/image/n5gVji)
*На данном экране отображены сгенерированные пары ключей RSA, поля для ввода зашифрованного текста и соответствующие кнопки управления. Функция "Генерация ключей" позволяет создавать новые криптостойкие пары для обеспечения безопасности.*
